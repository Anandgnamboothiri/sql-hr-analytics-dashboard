HR Analytics SQL Dashboard

This project focuses on analyzing HR data containing over 14,000 employee records using PostgreSQL and Power BI. Advanced SQL queries were used to explore workforce trends, employee attrition, department performance, salary distributions, and employee rankings. The results were integrated into an interactive Power BI dashboard to provide actionable insights for HR decision-making and workforce planning.

Key Features

Employee Attrition Analysis
Department Performance Analysis
Salary Distribution Insights
Employee Ranking using Window Functions
Interactive Power BI Dashboard
Business-Oriented KPI Reporting
